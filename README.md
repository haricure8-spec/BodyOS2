# BodyOS v3.0（受付・問診・BodyOS）

今回の実装範囲は次の3段階です。

1. 受付：患者iPhoneのQR受付／院のiPad直接入力
2. 問診：現在使用中の問診票を、院長が会話しながら自由記載できる形で電子化
3. BodyOS：命式・五行・二十四節気・UGLAB・養生の既存機能

## 起動方法（QR受付を使う場合）

1. Windows PCで `start-bodyos.bat` をダブルクリック
2. 初回だけ必要な部品を自動インストールします（インターネット接続が必要）
3. ブラウザにBodyOSが開きます
4. 受付欄に表示されたQRを、同じWi-Fiにつながった患者さんのiPhoneで読み取ります
5. 患者さんが送信すると、院内画面の「受付待ち」に表示されます
6. 「問診を開始」を押すと個人情報が問診カルテへ入ります

## iPadで直接入力する場合

院内画面の「院のiPadで直接入力」を押し、患者情報欄に入力します。

## index.htmlを直接開いた場合

命式・問診・BodyOSは利用できますが、端末間通信が必要なQR受付は使えません。

## 注意

- QR受付は院内の同じWi-Fi内で使う初期版です。
- 受付データはこのフォルダ内の `clinic_data/reception.json` に保存されます。
- 外部公開・本部同期・複数院対応はまだ入れていません。
