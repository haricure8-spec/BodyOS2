- UG6施術前/施術後
- セルフケア自動提案
- 次回確認自動提案
- 研究モード

- 同一患者継続記録
- JSON出力
- CSV出力
- この患者のデータ削除
- 全研究データ削除
- カルテA4
- 患者レポートA4
